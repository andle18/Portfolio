# Andrés Gomes — Frontend Portfolio

This is my personal frontend portfolio as a junior developer. It exists to showcase clean UI, responsive layouts, accessibility-first patterns, and practical frontend architecture in a real project.

## Features

- Clean, consistent UI design
- Fully responsive layouts across devices
- Smooth animations with Framer Motion
- Accessibility-minded structure and semantics
- Simple, maintainable component architecture

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router
- Vite
- Firebase (basic integration)

## Getting Started

```bash
npm install
```

```bash
npm run dev
```
